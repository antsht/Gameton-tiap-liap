# Инструкция по запуску

Проект состоит из двух частей: Backend (Go) и Frontend (HTML/JS/CSS).

## 1. Запуск Backend (Бот и API)

Перейдите в директорию `backend` и выполните команду запуска:

```bash
cd backend
go run ./cmd/app
```

Бэкенд-сервер запустится и будет доступен по адресу `http://localhost:8080`.

## 2. Запуск Frontend (Дашборд)

Frontend представлен обычными статическими файлами без сложных сборщиков.

Вы можете запустить его двумя способами:

**Способ А: Просто открыть файл (Самый простой)**
1. Зайдите в папку `frontend`
2. Откройте файл `index.html` в любом браузере (двойной клик или перетаскивание).

**Способ Б: Через локальный веб-сервер (Рекомендуется)**
Если у вас есть Python или вы используете VS Code с расширением Live Server:
```bash
cd frontend
# Для Python 3:
python -m http.server 3000
```
После этого откройте в браузере `http://localhost:3000`.
