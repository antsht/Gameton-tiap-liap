🧠 1. ГДЕ В ЭТОЙ ИГРЕ ДЕНЬГИ (очки)

Из правил:

Основные источники:
Терраформация (ОСНОВА)
обычная клетка: 10 очков за 1%
усиленная (x,y кратны 7): 15 очков за 1%
при TS=5 →
обычная: 50/ход
усиленная: 75/ход
Диверсия
даёт очки = как будто ты сам терраформил клетку
Бобры
дают x10 от клетки → это огромный буст
⚠️ 2. КЛЮЧЕВАЯ ИДЕЯ ИГРЫ

👉 Очки = скорость накопления терраформации × количество активных плантаций

НО:

лимит плантаций
деградация
потеря ЦУ = -5% очков + вайп

👉 Значит:

игра = экспансия + удержание сети + контроль риска

🧮 3. МАТЕМАТИКА ОПТИМУМА

Цель:

Maximize: Σ (TS × cell_value × uptime)

Где:

TS = 5 (база)
cell_value = 10 или 15
uptime = сколько ходов плантация живёт и работает
🔥 4. ГЛАВНЫЙ АЛГОРИТМ ХОДА
Каждый ход ты должен делать:
ШАГ 1 — Проверка критических рисков
если ЦУ под угрозой → ВСЁ бросить → защита
если есть изолированные плантации → восстановить связь

👉 Потеря ЦУ = катастрофа

ШАГ 2 — Проверка ROI (окупаемости действий)

Для каждой плантации считаешь:

ROI стройки = (ожидаемые очки от новой клетки) / (время постройки)

👉 если ROI высокий → строим
👉 если нет → делаем диверсию или усиливаем сеть

ШАГ 3 — Приоритет действий
🟢 1. Захват усиленных клеток (x%7 == 0)

ВСЕГДА приоритет №1

🟢 2. Расширение сети (экспансия)

Но только если:

есть связь с ЦУ
не превышен лимит
🔵 3. Оптимизация сети
сокращение расстояний (чтобы не терять эффективность)
перенос ЦУ ближе к центру
🔴 4. Диверсии (ситуативно)

Только если:

цель почти уничтожена
или это усиленная клетка
🟣 5. Бобры

Приоритет высокий, если:

рядом
можно добить
⚙️ 5. МЕГА-ВАЖНАЯ МЕХАНИКА (90% игроков ее игнорируют)
❗ Потеря эффективности через цепочку

Каждый переход:

CS / RS / SE / BE - 1

👉 значит:

❌ нельзя строить издалека
✅ нужно строить "фронтом"

🧠 6. ИДЕАЛЬНАЯ СТРАТЕГИЯ (ПО СТАДИЯМ)
🟢 РАННЯЯ ИГРА (0–150 ходов)

Цель:

максимально быстро разрастись
Алгоритм:
строишь вокруг ЦУ
ищешь клетки кратные 7
не уходишь далеко (сохраняешь связность)

👉 ошибка:

слишком длинные цепочки
🟡 СРЕДНЯЯ ИГРА (150–400)

Цель:

максимизация дохода
Алгоритм:
захват всех выгодных клеток
перенос ЦУ ближе к центру
контроль лимита плантаций

👉 начинаешь:

диверсии на сильных игроков
🔴 ЛЕЙТ (400–600)

Цель:

удержание и добивание
Алгоритм:
убиваешь слабые плантации врагов
защищаешь свои
добиваешь бобров
💣 7. ИМБА-ФИШКИ (САМОЕ ВАЖНОЕ)
🔥 1. Усиленные клетки = главный приоритет
1 усиленная клетка = 1.5 обычных
🔥 2. Массовая стройка

Несколько плантаций → 1 стройка

👉 ускоряет экспансию

🔥 3. Контроль лимита

Если превысишь:

удалится САМАЯ СТАРАЯ

👉 можно случайно удалить ЦУ (!!)

🔥 4. ЦУ = мобильный центр

Переноси:

ближе к фронту
но не рискуй
🔥 5. Тайминг разрушения

Если ты:

нанес последний удар → получаешь очки

👉 добивай всегда

🔥 6. Терраформация не вечная

После 40 ходов:

клетка деградирует

👉 значит:

выгодно возвращаться
🧠 8. ОПТИМАЛЬНЫЙ BOT-АЛГОРИТМ

Вот как должен думать бот:

for each turn:

    update game state

    if CU in danger:
        defend

    reconnect isolated plantations

    targets = find high value cells (x % 7 == 0)

    for plantation:
        if can expand efficiently:
            build

    for enemy plantation:
        if low HP and high value:
            sabotage

    for beaver:
        if can finish:
            attack

    if upgrade available:
        pick best upgrade

🧠 1. ЧТО ДЕЙСТВИТЕЛЬНО ДАЁТ ТОП-1

Важно понять:

👉 Побеждает НЕ тот, кто:

строит больше всего
атакует больше всего

👉 Побеждает тот, кто максимизирует:

(очки за ход) × (время жизни сети)
💥 Ключ:
1 потеря ЦУ = минус 5% + вайп сети
1 неправильная экспансия = -20–40% эффективности

👉 Значит:

игра = стабильный поток очков без обнулений

❌ 2. ПОЧЕМУ 90% ПРОИГРЫВАЮТ
Ошибка №1 — “жадная экспансия”
растягивают сеть
теряют связь → плантации умирают
Ошибка №2 — игнор деградации
клетки через 40 ходов начинают умирать
👉 теряют бесплатные очки
Ошибка №3 — плохая логистика
длинные цепочки → эффективность = 0
Ошибка №4 — игнор тайминга
не добивают врагов
теряют очки за диверсии
Ошибка №5 — поздние апгрейды
не бустят экономику вовремя
🏆 3. СТРАТЕГИЯ ТОП-1 (ЯДРО)
📌 Основной принцип:

Строй компактную, плотную, масштабируемую сеть вокруг high-value клеток и постоянно поддерживай 100% загрузку плантаций

🔥 4. МАКРО-СТРАТЕГИЯ (ПО ФАЗАМ)
🟢 ФАЗА 1: EXPANSION (0–120)

Цель:

быстро создать экономику
Делай:
строишь в плотном квадрате
ищешь клетки (x % 7 == 0 && y % 7 == 0)
НЕ делай:
длинные линии
одиночные плантации
🟡 ФАЗА 2: OPTIMIZATION (120–350)

Цель:

максимизировать доход
Делай:
переносишь ЦУ ближе к центру сети
перераспределяешь плантации
убираешь слабые клетки
🔴 ФАЗА 3: DOMINATION (350–600)

Цель:

удержать лидерство
Делай:
точечные диверсии
добивание слабых
контроль бобров
⚙️ 5. АЛГОРИТМ, КОТОРЫЙ ДАЁТ ТОП-1
🧠 Шаг 1 — Оценка клетки
cell_score =
    (is_boosted ? 1.5 : 1.0)
    × proximity_to_network
    × safety
🧠 Шаг 2 — Оценка действия

Для каждого действия:

action_score =
    (expected_points_gain)
    - (risk_cost)
    - (efficiency_loss)
🧠 Шаг 3 — Приоритет
Защита ЦУ
Поддержание связности
Захват усиленных клеток
Строительство рядом
Диверсия (если kill possible)
Бобры (если добивание)
⚡ 6. СЕКРЕТЫ ТОП-1 (САМОЕ ВАЖНОЕ)
💣 1. Плотная сеть > большая сеть
лучше 10 эффективных плантаций
чем 20 с потерями эффективности
💣 2. Всегда строй “локально”

👉 правило:

distance ≤ 2–3 от активной зоны
💣 3. Всегда держи 0 idle плантаций

Каждая плантация должна:

строить
чинить
атаковать
💣 4. Ротация клеток

После 100%:

👉 плантация исчезает → ты должен:

заранее строить следующую
💣 5. Контроль деградации

Через 40 ходов:

👉 возвращайся на старые клетки

💣 6. Kill-steal механика

👉 ВСЕГДА:

отслеживай HP врагов
добивай
💣 7. Апгрейды (критично)
Лучший порядок:

🤖 7. ИДЕАЛЬНЫЙ BOT (КАК ПИСАТЬ)
Каждый ход:
update_state()

handle_emergency():
    protect CU
    reconnect network

targets = find_best_cells()

for plantation:
    if can_build_high_value:
        build
    else if can_finish_enemy:
        sabotage
    else if can_expand:
        build
    else:
        support (repair / reposition)

handle_beavers()

apply_upgrade()
